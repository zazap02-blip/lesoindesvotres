Voici un guide complet pour créer votre boîte mail pro sur IONOS :

Étape 1 — Choisir une offre

IONOS propose deux formules principales :

  • Mail Basic : 2 Go par boîte, idéal pour les indépendants ou petites structures
  • Mail Business : 50 Go par adresse, avec agenda intégré et synchronisation MobileSync Cercle Entreprises

Les tarifs démarrent à partir de 1 €/mois par boîte email, avec un domaine gratuit la première année. Tdrgroupe

Étape 2 — Créer votre adresse mail

Une fois connecté à votre compte IONOS, voici la marche à suivre :

  1. Allez dans Menu, puis cliquez sur la tuile Email
  2. Sélectionnez le contrat que vous souhaitez utiliser
  3. Cliquez sur Créer une nouvelle adresse email
  4. Choisissez le type d'email (ex. : Email Pro | 50 GB)
  5. Dans le champ Email, saisissez la partie avant le @, puis sélectionnez votre domaine
  6. Saisissez votre mot de passe
  7. (Facultatif) Activez une redirection vers une autre adresse existante
  8. (Facultatif) Activez la protection anti-spam et/ou antivirus premium
  9. Cliquez sur Enregistrer IONOS Assistance

Étape 3 — Accéder à votre boîte mail

Vous pouvez accéder à vos mails de plusieurs façons :

  • Webmail : directement sur webmail.ionos.fr depuis n'importe quel navigateur
  • Client mail (Outlook, Thunderbird) : en configurant les paramètres IMAP/SMTP
  • Smartphone : en ajoutant le compte dans votre application mail

Si vous utilisez Gmail, vous pouvez aussi configurer une redirection automatique et ajouter votre adresse IONOS comme alias d'expédition, via le serveur SMTP smtp.ionos.fr sur le port 465 (SSL). JCEN Web

Bonnes pratiques

  • Activez l'authentification à deux facteurs (2FA) pour sécuriser votre compte Mailbird
  • Ajoutez une signature professionnelle avec vos coordonnées
  • Configurez un répondeur automatique pour les absences

Avez-vous déjà un compte IONOS et un nom de domaine, ou partez-vous de zéro ?
